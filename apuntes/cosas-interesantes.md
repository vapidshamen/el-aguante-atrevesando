# cosas-interesantes

## puentes linguisticos

interesantes puentes linguisticos entre el espanol y el griego:
1. ἕνα (ena) es acusativo masculino singular para el numero "uno": "a/al uno" (hablando {sobre el | en dirreccion al} uno). me gusta q la palabra griega aparece a "eva" (la primera mujer creada para el/adan -- creada al uno)
2. δὲ (de) significa "sobre (la parte previa)" o "perteniciendo a la parte siguida"; en muchos sitios se puede leer como "de" en espanol -- como *concepto* de *concepto*, pero no se usa "de" para significar posesion (como "zapatos de el"), sino el caso genetivo (como "zapatos autos" (autos es genetivo masculino singular: "de el"))
3. ἐν (en) lo mismo q "en" (espanol) o "in" (ingles): "dentro de"
