# version **muy literal**

#### en esta traduccion
- no intento tener gramatica perfecta
- intento seguir los casos/tensos de cada palabra en el texto original

#### notas
- el griego usa la forma "X de ella/ellas/el/ellos" para decir posession -- y traducido literalmente asi, es un poco confuso, entonces muchas veces lo cambio a "su/sus X" o "X (de aquello anterior)"
- a veces, pienso yo q la punctuacion puesta en el texto esta mal puesto (xj, [TODO - santiago 2])
- a veces, pienso yo q el verso empieza en medio de una frase o quita la primera palabra de una frase (xj, [TODO - heb 3:1-2])
- he notado q "δὲ" usualmente es reflexivo (calificando la parte anterior con la siguiente) -- entonces lo traduzco como "asi" o "asimismo"


## razonamiento
- "dios" vs "el dios"
	- este tema es un poco dificil descodificar xq en tiempos antiguos, "dios" (sin articulo) puede ser:
		- uno de su panteon (apollo, dionisio, zeus)
		- una persona muy superior (como un emperador romano)
	- el caso vocativo quita el articulo 
	- a veces
